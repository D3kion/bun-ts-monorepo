# mono-ts

To install dependencies:

```bash
bun install
```

To run:

```bash
bun dev
```

To prepare change for changelog:

```bash
bun changelog
```

To release (bump version and make changelog):

```bash
bun release
```

This project was created using `bun init` in bun v1.1.40. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
